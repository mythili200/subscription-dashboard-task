# Subscription Management Dashboard (MERN + MUI)

This repository contains a sample MERN project (backend + frontend) for a subscription management dashboard.

## Quick start - Backend
1. cd backend
2. npm install
3. copy .env.example to .env and fill values
4. npm run seed
5. npm run dev

## Quick start - Frontend
1. cd frontend
2. npm install
3. copy .env.example to .env and set VITE_API_URL (e.g. http://localhost:5000/api)
4. npm run dev
